Docker Swarm

Positives

  • Built into the Docker runtime
  • Easy to setup
  • Simple, opinionated, full-stack model

Negatives

  • Arrived late
  • Slow to evolve
  • Unstable networking
  • Not adopted by vendors

Where we're headed

Kubernetes

  • Open sourced in 2015
  • Maintained by the CNCF
  • General-purpose approach to container orchestration
  • Very wide adoption by vendors

Roadmap items

  • Finish Vault implementation
  • Support Windows-based containers
  • Get environment approved for critical/restricted data
  • Spinnaker - Continuous delivery
  • Helm - Shareable Kubernetes resources
  • Container Storage Interface
  • Log management strategy
  • Investigate service meshes
